# Rust Bank
> A fake banking program written for an introductory Rust learning.

## Features
Rust Bank uses these Rust concepts:
1. Crates
1. Importing from other files
1. Structs & Implementations
1. The type system
1. Functions
1. Loops
1. `match` statements
1. Conditionals
1. Vectors
1. Ownership & Borrowing
1. `stdout` and `stdin`
