# Rust Bank
> A fake banking program written for an introductory Rust learning.

## Features
Rust Bank uses these Rust concepts:
1. Crates
2. Importing from other files
3. Structs & Implementations
4. The type system
5. Functions
6. Loops
7. `match` statements
8. Conditionals
9. Vectors
10. Ownership & Borrowing
11. Printing
